# # portafolio roand 

en este proyecto quise desarrollar mi portafolio de una manera divertida; juntando el estilo de fallout y yo haciendo una fusion que siento que quedo divertida; el proyecto se segmenta por tres partes:

## estructura

- WelcomeScreen: es la que nos da la bienvenida a la aplicacion y luego nos da paso al portafolio.
- PortafolioScreen: dentro del portafolio desarrollo las siguientes preguntas: quien soy; mis hobbies y que hago.
- ContactScreen: luego abajo se puede acceder a mi informacion de contacto y skills (habilidades).

es un proyecto bastante simple que desarrolle; pero me gusto como quedo con los agregados esteticos y como la estructura simple tomo una forma chevere.
